# DATA

## データソース
| ソース | 役割 |
|--------|------|
| NOAA CRW | DHW閾値の基準 |
| NASA MUR | 日次SST（1km、22年分） |
| モニ1000 | 実測白化率（将来） |
| Stormglass | 海況予報 + 雲量 |

衛星DHWは現地の2〜3倍高く出る。「目安」として提示。

- **耐性**: Porites（これが白化したら深刻）
